# 小豆子旅行轨迹 - 阶段4：页面开发

**项目名称：** 小豆子旅行轨迹
**阶段名称：** 页面开发
**生成时间：** 2026-03-11
**版本：** v1.0

---

## 📐 阶段概述

本阶段主要完成6个页面的开发，包括：
1. index - 首页
2. checkin - 打卡页
3. footprint - 足迹页
4. list - 清单页
5. timeline - 时间线页
6. checkin-list - 打卡列表页

**任务数量：** 6个
**预计时间：** 5天（2400分钟）
**前置条件：** 阶段1-3完成

---

## 📋 开发任务列表

### 任务4.1：index - 首页开发

**任务描述：**
开发首页，展示旅行统计和最近打卡记录。

**主要功能：**
- 显示旅行统计数据（旅行次数、城市、博物馆、游乐场、本月旅行）
- 显示最近3次打卡记录
- 跳转到打卡列表页
- 跳转到打卡页
- 下拉刷新统计

**验收标准：**
- [ ] 统计数据展示正确
- [ ] 最近打卡显示正确（最多3条）
- [ ] 页面跳转正常
- [ ] 下拉刷新正常
- [ ] 页面样式美观
- [ ] 性能满足要求（加载时间 < 2秒）

**预计时间：** 60分钟

---

### 任务4.2：checkin - 打卡页开发

**任务描述：**
开发打卡页面，支持新增打卡记录。

**主要功能：**
- 选择/搜索场所（博物馆/游乐场）
- 使用当前位置匹配（GPS定位）
- 填写参观日期（日期选择器）
- 填写参观笔记（最多500字）
- 上传图片（最多9张，自动压缩）
- 提交打卡
- 表单验证

**验收标准：**
- [ ] 场所选择功能正常
- [ ] 当前位置定位正常
- [ ] 日期选择功能正常
- [ ] 笔记输入正常（字数限制）
- [ ] 图片上传正常（最多9张）
- [ ] 图片压缩正常（大图自动压缩）
- [ ] 打卡提交成功
- [ ] 表单验证正确（必填项校验）
- [ ] 错误提示友好
- [ ] 页面样式美观

**预计时间：** 70分钟
