### 任务7.2：提交审核

**任务描述：**
将小程序提交到微信审核，准备上线。

**准备项：**
1. 填写审核说明
2. 上传审核截图
3. 准备测试账号
4. 准备测试流程说明
5. 填写隐私政策
6. 配置服务类目

**提交材料：**

**审核说明：**
```
小程序名称：小豆子旅行轨迹
小程序简介：记录旅行足迹，发现美好世界
小程序分类：旅行 / 生活 / 工具

功能说明：
- 旅行打卡记录
- 足迹地图展示
- 博物馆/游乐场清单
- 时间线回顾
- 想去收藏
```

**隐私政策：**
```
隐私政策

1. 信息收集
   - 用户授权信息（微信 OpenID）
   - 用户主动提供的打卡数据（场所、日期、笔记）
   - 用户主动上传的打卡图片

2. 信息使用
   - 用于保存用户的打卡记录
   - 用于展示用户的旅行足迹
   - 用于搜索和筛选场所

3. 信息存储
   - 用户数据存储在微信云开发数据库
   - 用户图片存储在微信云存储
   - 不存储任何敏感信息（如身份证号、银行卡号等）

4. 信息共享
   - 用户数据仅在用户授权下使用
   - 不会向第三方共享用户数据
   - 不会出售用户数据

5. 信息保护
   - 使用微信官方云开发服务
   - 采用 HTTPS 加密传输
   - 数据库权限控制严格

6. 用户权利
   - 用户可以查看自己的数据
   - 用户可以删除自己的数据
   - 用户可以注销账号
```

**服务类目：**
```
旅行 / 生活 / 工具
```

**验收标准：**
- [ ] 审核说明填写完整
- [ ] 隐私政策填写完整
- [ ] 服务类目配置正确
- [ ] 测试账号可用
- [ ] 测试流程说明清晰
